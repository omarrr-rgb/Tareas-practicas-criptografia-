# Cifrado César en Python

Este repositorio contiene un programa que implementa el **cifrado César**, desarrollado en **Python**.

## Funcionamiento
El programa se ejecuta desde la **terminal**.  
Al iniciar la ejecución, el sistema solicita al usuario que **ingrese el texto que desea descifrar**.  
Una vez ingresado el texto, el programa procesa la información y **muestra el resultado descifrado en pantalla**.

## Características del código
- El cifrado está implementado completamente en **Python**.
- El código distingue entre **letras mayúsculas y minúsculas**.
- Se consideran únicamente los **caracteres del abecedario**, evitando afectar otros símbolos.
- El manejo del texto se realiza mediante distintas secciones del código para asegurar un funcionamiento correcto del cifrado y descifrado.
